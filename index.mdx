## What is Codewolf?

Codewolf provides AI agents that work directly in Slack to handle production engineering tasks. Tag <span style={{color: '#07C983'}}>@Codewolf</span> and agents investigate incidents, optimize costs, and help you ship faster with production context.

## Key features

**Investigate incidents**
AI agents analyze your code, infrastructure, and telemetry to find root causes and deliver fixes with pull requests.

**Monitor continuously**
Agents watch for issues, review PRs, run post-deployment checks, and catch problems before they escalate.

**Automate workflows**
Set up scheduled tasks like daily summaries, weekly reports, and automated ticket creation.

**Optimize costs**
Discover savings opportunities across your infrastructure and services.

## Security

Your data is never shared with other customers or used for model training. Codewolf uses:
- Isolated agent workspaces
- AES-256 encryption
- SSO & SAML support
- SOC 2 Type II certification
